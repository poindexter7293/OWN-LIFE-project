# OWN LIFE

**OWN LIFE**는 사용자의 **운동 기록, 식단 기록, 체중 및 목표 관리**를 한곳에서 관리할 수 있도록 만든 **헬스케어 웹 서비스**입니다.  
사용자는 하루 동안 섭취한 음식과 운동 내역을 기록하고, 이를 바탕으로 **칼로리 및 영양소 변화**를 시각적으로 확인할 수 있습니다.  
또한 자유게시판과 회원 관리 기능을 통해 커뮤니티와 개인 맞춤형 관리 기능을 함께 제공하는 것을 목표로 하였습니다.

본 프로젝트는 **Spring Boot 기반 MVC 구조**로 설계되었으며, **Thymeleaf 템플릿 엔진**, **Spring Data JPA**, **MySQL**을 활용하여 구현하였습니다.

```text
ownlife

│  ├─ controller

│  └─ ...
└─ pom.xml
```

### 패키지 구성

- **controller** : 요청 처리 및 화면 이동
- **service** : 비즈니스 로직 처리
- **repository** : DB 접근
- **entity** : JPA 엔티티 정의
- **dto** : 화면 및 데이터 전달 객체
- **templates** : Thymeleaf 화면 템플릿
- **static** : CSS, JavaScript 정적 자원
- **db** : 테이블 생성 SQL 및 DB 구조 파일

## 6. Main Features

### 6-1. 메인페이지

- 체중, 섭취 칼로리, 소모 칼로리, 영양소 데이터를 한눈에 볼 수 있는 대시보드 제공
- 그래프를 통해 일간 / 주간 / 월간 / 연간 변화 확인
- 체크박스를 통한 그래프 항목 ON/OFF 구성
- 금일 영양소 섭취 비율(탄수화물, 단백질, 지방) 시각화
- 사용자의 기록 현황을 빠르게 확인할 수 있는 메인 화면 구성

### 6-2. 식단 관리

- 날짜별 식단 기록 조회
- 캘린더 기반 식단 입력 및 조회
- 음식 데이터 기반 칼로리 / 탄수화물 / 단백질 / 지방 계산
- 하루 섭취량 요약 제공
- 목표 섭취 칼로리 대비 현재 섭취 현황 표시
- 일 / 주 / 월 / 년 단위 차트 제공
- 사용자가 직접 음식 정보를 추가하여 기록 가능한 구조 반영
- 향후 공공데이터 또는 검색 기능을 활용한 음식 영양정보 보완 가능

### 6-3. 운동 일지

- 날짜별 운동 기록 조회
- 운동 종류에 따른 빠른 입력 기능 제공
- 횟수형 운동, 시간형 운동, 거리형 운동 기록 지원
- 운동량에 따라 소모 칼로리 자동 계산
- 직접 입력 운동 추가 기능 제공
- 운동 기록 차트 시각화 제공
- 경로 기반 운동 기록을 저장할 수 있는 구조 설계
- 운동 자세 안내용 유튜브 영상 연동 확장 가능

### 6-4. 자유게시판

- 게시글 목록 조회
- 게시글 상세 조회
- 게시글 작성 / 수정 / 삭제
- 댓글 작성 / 삭제
- 사용자 간 간단한 커뮤니티 기능 제공
- 관리자 권한 기반 관리 가능

### 6-5. 회원 기능

- 회원가입
- 로그인 / 로그아웃
- 아이디, 이메일, 닉네임 중복 검사
- 마이페이지 정보 수정
- 회원탈퇴
- 일반 로그인과 소셜 로그인 구조 지원

### 6-6. 관리자 기능

- 관리자 전용 회원 관리 페이지 제공
- 회원 검색 및 상태 조회
- 회원 상태 변경 기능 제공

## 9. Database Design

프로젝트 폴더 내 `db` 디렉터리 기준으로 다음과 같은 주요 테이블이 구성되어 있습니다.

| 테이블명 | 설명 |
|---|---|
| member | 회원 정보 |
| social_account | 소셜 계정 연동 정보 |
| member_goal_history | 목표 변경 이력 |
| food | 음식 영양 정보 |
| meal_log | 식단 기록 |
| exercise_type | 운동 종류 기준 정보 |
| exercise_log | 운동 기록 |
| activity_route_log | 경로 기반 운동 기록 |
| board_post | 게시글 |
| board_comment | 댓글 |
| daily_summary | 일별 요약 정보 |
| weight_log | 체중 기록 |

### 핵심 DB 설계 포인트

- **member** : 회원 기본 정보, 목표 칼로리, 목표 체중, 로그인 타입 관리
- **social_account** : 네이버 / 카카오 / 구글 계정 연동 관리
- **food / meal_log** : 음식 정보와 식단 기록 분리
- **exercise_type / exercise_log** : 운동 기준 데이터와 실제 운동 기록 분리
- **activity_route_log** : 거리 기반 운동의 경로 정보 저장
- **board_post / board_comment** : 자유게시판 기능 구성
- **daily_summary / weight_log** : 사용자 건강 데이터 요약 및 체중 기록 관리
